## 기능정의


- - -
### Main - 로그인 안 한 메인 화면
1. 로고를 클릭하면 메인창으로 이동한다.(메인창이어도 다시 새로고침)
2. 로그인창으로 이동한다.
3. 회원가입창으로 이동한다.
4. 사이트의 테마 애니메이션을 보여준다.
5. 실행시킬 코드를 작성한다.
6. 5번에 작성한 코드를 실행한 결과물을 출력한다.
7. 5번에 작성한 코드를 실행해준다.
8. 로그인 하지 않은 상태에서는 저장이 불가하므로, 로그인창으로 이동한다.

### Main - 로그인 한 메인 화면
1. 메인 페이지로 이동한다.
2. 사용자 코드 저장소인 코드 트리 창으로 넘어간다.
3. Q&A 창으로 넘어간다.
4. 퀴즈 창으로 넘어간다.
5. 랭킹 창으로 넘어간다.
6. 마우스를 갖다대면 마이페이지 tab이 드러난다.
7. 로그아웃을 한다.
8. 등급이 드러나는 창을 띄운다.
9. 실행시킬 코드를 작성한다.
10. 9번에 작성한 코드를 실행한 결과물을 출력한다.
11. 9번에 작성한 코드를 실행해준다.
12. 코드 트리에 저장할 위치를 선택할 수 있는 팝업창을 띄운다.
13. 코드 프레스트의 컨셉 애니메이션이 재생된다.
14. 정보 제공(기업 정보)
15. 페이지의 상단으로 이동시켜준다.
16. 왼쪽에 있는 애니메이션을 없애주고 그만큼 코드창을 확대해준다.
17. 사용자가 실행하고 싶은 언어를 선택할 수 있다.
18. 유저의 팔로워 수를 보여준다.
19. 유저의 팔로잉 수를 보여준다.

### User - 로그인
1. 이메일을 입력하는 부분이다.
2. 비밀번호를 입력하는 부분이다.
3. 클릭 시 1번과 2번에서 입력한 유저 정보가 맞는지 확인한다.
  3-1. 맞으면 로그인되어 메인 화면 페이지로 이동한다.
  3-2. 틀리면 페이지 이동 없이 어떤게 틀렸는지 나타난다.
4. 클릭 시 회원가입 페이지로 이동한다.
5. 비밀번호 찾기 페이지로 이동한다.

### User - 회원가입 - 임시 저장
1. 닉네임을 입력하는 부분이다.
2. 클릭 시 중복된 닉네임인지 확인한다.
3. 이메일을 입력하는 부분이다.
4. 클릭 시 중복된 이메일인지 확인한다.
5. 비밀번호를 입력한다.
6. 비밀번호를 확인 차 다시 입력한다.
7. 클릭 시 이메일 인증 페이지로 이동하고, 입력한 이메일로 인증 메일을 전송한다.
8. 회원 가입이 취소되고 메인 페이지로 이동한다.

### User - 회원가입 - 최종 저장
1. 사용자가 입력한 이메일로 온 메일의 인증 번호를 입력한다.
2. 클릭 시 이메일로 인증 번호를 재전송한다.
3. 클릭 시 인증 번호가 맞는지 확인한다.
  3-1. 맞으면 메인 페이지로 이동한다.
  3-2. 틀리면 페이지 이동 없이 인증 번호가 틀렸다고 나타난다.

### User - 아이디 찾기
1. 사용자가 가입했던 이메일을 입력한다.
2. 1번에 입력된 이메일로 비밀번호 재설정 링크를 보내준다.

### User - 비밀번호 재설정
1. 재설정할 비밀번호를 입력할 수 있다.
2. 재설정할 비밀번호를 다시 입력하여 확인한다.

- - -

### Code Tree - 1
1. 클릭 시 컨테이너 생성 페이지로 이동한다.

### Code Tree - 2
1. 새로운 패키지 및 파일을 생성할 수 있는 팝업창을 띄워준다.
2. 사용자가 원하는 언어를 선택할 수 있다.
3. 사용자가 저장한 파일들을 트리 형태로 보여준다.
4. 트리창 파일의 코드를 보여주고 수정할 수 있다.
5. 사용자가 선택한 파일을 실행시킨다.
6. 4번의 코드를 저장시켜준다.
7. 5번 실행 버튼을 클릭 시, 실행시킨 파일의 결과를 출력해준다.

### Q&A - 메인
1. 질문 언어의 종류를 카테고리로 나타내고 스크롤을 내려도 따라다닌다.
2. 검색의 종류를 선택한다.
  2-1. 번호
  2-2. 제목
  2-3. 글쓴이
  2-4. 제목 + 글쓴이
  2-5. 내용
  2-6. 댓글 내용
  2-7. 댓글 작성자
3. 검색할 내용을 입력한다.
4. 3번의 내용을 찾아 검색해준다.
5. 질문게시판 글의 리스트를 보여준다.
  5-1. 사용자가 검색한 글의 리스트를 보여준다.
6. 글작성 페이지로 이동한다.
7. 페이지를 이동한다.
8. 카테고리의 종류를 선택한다.

### Q&A - 질문 글(메인 5-1)
1. 질문글의 제목을 보여준다.
2. 질문글의 내용을 보여준다.
3. 질문글에 사용자들이 단 댓글을 보여준다.
4. 댓글의 내용을 작성할 수 있다.
5. 작성한 댓글을 등록한다.
6. 댓글을 단 사용자에 한해서 클릭 시 댓글이 삭제된다.
7. 자신이 단 댓글을 수정할 수 있다.
8. 자신이 쓴 질문글을 수정할 수 있다.
9. 자신이 쓴 질문글을 삭제할 수 있다.

### Q&A - 질문 작성(메인 6)
1. 언어를 선택할 수 있다.
2. 질문의 카테고리를 선택할 수 있다.
3. 질문의 제목을 쓸 수 있다.
4. 질문의 내용을 쓸 수 있다.
5. 질문 등록을 취소하고, 퀴즈의 메인 페이지로 이동한다.
6. 질문을 등록하고 퀴즈의 메인 페이지로 이동한다.

### Q&A - 질문 수정
1. 언어를 수정할 수 있다.
2. 질문의 카테고리를 수정할 수 있다.
3. 질문의 제목을 수정할 수 있다.
4. 질문의 내용을 수정할 수 있다.
5. 질문 수정을 취소하고, 퀴즈의 메인 페이지로 이동한다.
6. 질문을 수정하고 퀴즈의 메인 페이지로 이동한다.

### 랭킹
1. 기간별 랭킹 순위를 보여준다.
2. 선택한 기간의 랭킹 리스트가 나타난다.
3. 유저의 닉네임을 클릭 시 유저 정보 페이지로 넘어간다.

### 퀴즈 - 1
1. 사용자가 선택한 난이도 별로 문제를 보여준다.
2. 사용자가 선택한 언어 별로 문제를 보여준다.
3. 제목과 글쓴이별로 검색 범주를 선택할 수 있다.
4. 사용자가 검색하고 싶은 키워드를 입력할 수 있다.
5. 3번과 4번의 내용을 조합하여 검색할 수 있다.
6. 검색한 내용을 리스트로 나타내준다.
  6-1. 번호 : 고유번호를 사용한다.
  6-2. 제목 : 문제 제목을 보여준다.
  6-3. 정보 : 문제의 채점 여부를 알려준다.(성공/실패/미해결 : 안 보임)
  6-4. 글쓴이 : 문제의 작성자를 보여준다.
  6-5. 조회수 : 문제의 조회수를 보여준다.
  6-6. 추천수 : 사람들이 문제를 추천한 수를 보여준다.
7. 문제 작성 페이지로 이동한다.
8. 페이지를 이동한다.
9. 제목을 클릭할 시 해당 문제 페이지로 이동한다.

### 퀴즈 - 문제
1. 문제 번호를 알려준다.
2. 문제에 대한 질문 목록창으로 넘어간다.
3. 오타/오역에 대한 신고 목록창으로 넘어간다.
4. 문제 내용을 보여준다.
5. 입력 예제를 보여준다.
6. 출력 예제를 보여준다.
7. 문제를 풀 언어를 선택한다.
  7-1. C / C++ / Java / JavaScript / Python
8. 문제의 코드를 작성한다.
9. 8번의 코드를 실행시킨다.
10. 9번을 눌렀을 때 출력 결과를 보여준다.
11. 작성한 코드를 채점 결과창으로 보내준다.
12. 문제 작성자에게만 보여지고 클릭 시 문제 수정 창으로 이동한다.+

### 퀴즈 - 문제 제출
1. 클릭 시 퀴즈 메인 페이지로 이동한다.
2. 클릭 시 팝업창이 닫힌다.

### 퀴즈 - 문제 작성
1. 언어를 선택할 수 있다.
2. 난이도를 선택할 수 있다.
3. 제목을 입력할 수 있다.
4. 내용을 입력할 수 있다.
5. 예제를 입력할 수 있다.
6. 출력을 입력할 수 있다.
7. 문제 작성을 취소하고, 문제 목록 페이지로 이동한다.
8. 문제를 등록하고 문제 목록 페이지로 이동한다.

### 퀴즈 - 문제 수정
1. 언어를 수정할 수 있다.
2. 난이도를 수정할 수 있다.
3. 제목을 수정할 수 있다.
4. 내용을 수정할 수 있다.
5. 예제를 수정할 수 있다.
6. 출력을 수정할 수 있다.
7. 문제 수정을 취소하고, 문제 목록 페이지로 이동한다.
8. 문제를 수정하고 문제 목록 페이지로 이동한다.

### 퀴즈 - 문제 - 질문 메인
1. 문제의 제목을 나타낸다.
2. 질문 리스트가 나타난다.
3. 정렬 기준을 선택할 수 있다.
4. 페이지를 이동할 수 있다.
5. 클릭 시 질문 작성 페이지로 이동한다.
6. 검색 기준을 선택할 수 있다.
7. 검색키워드를 입력할 수 있다.
8. 7번에 입력된 검색키워드로 검색해준다.

### 퀴즈 - 문제 - 오타/오역 신고 메인
1. 문제의 제목을 나타낸다.
2. 오타/오역 신고 리스트가 나타난다.
3. 정렬 기준을 선택할 수 있다.
4. 페이지를 이동할 수 있다.
5. 클릭 시 오타/오역 신고 작성 페이지로 이동한다.
6. 검색 기준을 선택할 수 있다.
7. 검색키워드를 입력할 수 있다.
8. 7번에 입력된 검색키워드로 검색해준다.

### 퀴즈 - 정답자
1. 언어를 선택할 수 있다.
2. 정답자 리스트가 나타난다.
3. 공개 유저면 링크 활성화로, 클릭 시 정답자의 작성 코드 페이지로 이동한다.
4. 비공개 유저면 링크 비활성화로, 코드 페이지로 이동할 수 없다.
5. 3번의 활성화된 닉네임 링크 클릭 시, 그 사용자의 정답 코드가 나타난다.
6. 클릭 시 유저 정보, 코드 보기로 2가지 선택이 가능하다.
  6-1. 유저 정보 클릭 시 해당 유저의 정보 페이지로 이동한다.
  6-2. 코드 보기 클릭 시 페이지 하단에 그 유저의 정답 코드가 나타난다.

### 마이페이지 - 타임 라인
1. 로고를 클릭하면 메인창으로 이동한다.
2. 사용자의 등급을 나타내준다.
3. 사용자 닉네임 클릭 시 마이페이지로 이동한다.(마이페이지여도 다시 이동)
4. 로그아웃 후 로그인 안 한 메인 페이지로 이동한다.
5. 사용자의 타임라인을 보여준다.
6. 사용자의 계정 관리 페이지로 이동한다.
7. 사용자의 팔로우 관리 페이지로 이동한다.
8. 사용자의 코드트리 관리 페이지로 이동한다.
9. 사용자의 문제 내역 및 관리 페이지로 이동한다.
10. 사용자 및 팔로우된 유저들의 문제 내역들을 보여준다.
11. 다음 페이지로 구성되어 있지 않고, 스크롤을 내리면 다음 리스트들이 자동으로 추가된다.

3. 버튼 클릭 시 닉네임 변경창이 나타난다.
4. 버튼 클릭 시 비밀번호 변경창이 나타난다.
5. 클릭 시마다 계정 공개 여부를 공개 / 비공개로 바뀐다.
6. 회원 탈퇴할 수 있는 창이 나타난다.

### 마이페이지 - 계정 관리 3의 변경창

4. 닉네임을 변경해준다.

5. 요청을 삭제한다.

### 마이페이지 - 팔로우 관리 - 팔로워,팔로잉 5번의 삭제 팝업
1. 요청을 삭제할 수 있는 알림창을 보여준다.
2. 삭제를 취소하고 팝업창을 닫는다.
3. 요청 삭제를 진행하고 팝업창을 닫는다.

### 마이페이지 - 코드트리 관리
1. 컨테이너의 이름을 보여준다.
2. 컨테이너를 삭제할 수 있는 팝업창이 뜬다.
3. 컨테이너의 언어를 보여준다.
4. 해당 컨테이너의 코드트리창으로 이동한다.
5. 새로운 컨테이너를 생성하는 창으로 이동한다.

### 마이페이지 - 코드트리 관리 - 컨테이너 생성
1. 컨테이너의 이름을 입력한다.
2. 컨테이너 생성을 취소한다.
3. 컨테이너를 생성하고 코드트리 관리 메인 페이지로 이동한다.
  3-1. 컨테이너의 이름이 이미 존재할 시 생성이 되지 않고 중복된 컨테이너 이름이라고 안내된다.

### 마이페이지 - 코드트리 관리 - 2번의 팝업창
1. 컨테이너 삭제 팝업창이다.
2. 삭제 취소로 팝업창을 닫는다.
3. 컨테이너 삭제를 하며 팝업창을 닫는다.

### 마이페이지 - 문제 관리
1. 작성한 문제의 번호를 보여준다.
2. 작성한 문제의 제목을 보여준다.
3. 작성한 문제의 조회수를 보여준다.
4. 작성한 문제의 추천수를 보여준다.
5. 작성한 문제를 삭제할 수 있는 팝업창이 뜬다.

7. 페이지의 상단으로 이동한다.
